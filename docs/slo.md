# SLO定義書: Biblivote

**バージョン**: 1.0
**有効期間**: 2026-04-15（投票開始）〜 2026-05-15（投票終了）
**オーナー**: 井上裕介（cyokozai）

> 注: Biblivoteは1ヶ月限定のイベント向けシステムであり、厳密なSLA契約は存在しない。
> ここでは「投票期間中に参加者が不満を感じない品質」を目標として設定する。

---

## CUJ-1: ユーザーが投票を完了できる（最重要）

投票フォームを開いてから送信完了画面に到達するまでのジャーニー。

### SLI定義
- **指標**: 可用性（フロントエンド + GAS）
- **計算式**: 投票送信成功数 / 投票送信試行数（5xxエラーを除く）
- **データソース**: GAS logsシートの `result` 列（success / error カウント）
- **測定窓**: 投票期間全体（2026-04-15〜05-15）
- **除外条件**: ユーザーのネットワーク障害によるタイムアウト（クライアント起因）

### SLO目標値
- **目標**: 99.0%
- **根拠**: GitHub Pages / Cloudflare Pages の標準可用性は99.9%以上。GASの制限（実行時間・同時接続）が律速要因となるため、余裕を持たせた99.0%を目標とする。

### エラーバジェット
- **投票期間31日のバジェット**: 約7.4時間相当の障害
- **アラートポリシー**:
  - Warning: logsシートの`error`が連続10件発生した場合 → 手動確認
  - Critical: logsシートの`error`率が10%を超えた場合 → GAS設定を確認

## CUJ-2: 書籍検索が応答する

Q4・Q5でユーザーがタイトルを入力してからサジェストが表示されるまでのジャーニー。

### SLI定義
- **指標**: レイテンシ（Google Books API）
- **計算式**: レスポンス時間 < 1,000ms のリクエスト割合
- **データソース**: ブラウザのPerformance API（開発中は手動計測）
- **測定窓**: スポット計測（デプロイ時）
- **除外条件**: Google Books APIの障害（外部依存のため許容）

### SLO目標値
- **目標**: P95 < 1,000ms
- **根拠**: ユーザーリサーチによると入力から1秒以内のレスポンスが「快適」の閾値。

### エラーバジェット
- 外部APIのためエラーバジェット管理は行わない。障害時はサジェストを非表示にして手動入力にフォールバックする。

## CUJ-3: 投票結果ダッシュボードが閲覧できる

Grafanaダッシュボードにアクセスして最新の集計データが表示されるジャーニー。

### SLI定義
- **指標**: 可用性（Grafanaインスタンス）
- **計算式**: ダッシュボードページの正常表示率
- **データソース**: Grafana内蔵のヘルスチェック
- **測定窓**: 投票期間全体
- **除外条件**: 定期メンテナンス時間帯

### SLO目標値
- **目標**: 99.0%
- **根拠**: 結果閲覧は投票フォームよりも重要度が低いため、同等の99.0%を設定。

## エラーバジェットポリシー（CUJ-1 用）

| バジェット消費率 | アクション |
|--------------|---------|
| 0〜50% | 通常の開発・デプロイを継続 |
| 50〜75% | GASの実行ログを詳細確認。不正投票増加の可能性を調査 |
| 75〜100% | 新機能追加を停止。GASの安定性改善に集中 |
| 100%超過 | 投票フォームを一時停止し原因調査。SNSで状況をアナウンス |

---

## 監視・アラート設定

| 項目 | 確認方法 | 頻度 |
|------|---------|------|
| GAS実行ログ | Google Apps Scriptコンソール → 実行ログ | 毎日1回（投票期間中） |
| logsシートのエラー率 | スプレッドシートの`result`列をフィルタ | 毎日1回 |
| 投票数の異常増加 | votesシートの行数を確認（不正投票検知） | 毎日1回 |
| GAS使用量 | Googleアカウント → Apps Script割り当て | 週1回 |

> 本プロジェクトのスコープ上、Prometheusなどの監視ツールは導入しない。
> スプレッドシートとGASコンソールを主要なオブザーバビリティツールとして使用する。
